This project predicts whether a credit card transaction is fraudulent or genuine using machine learning. It includes data preprocessing, model training, evaluation, and explainability.

### Files
- `fraud.py` – Basic ML model  
- `fraud_advanced.py` – Improved model with SMOTE  
- `fraud_app.py` – Predicts fraud for new inputs  
- `shap_summary.png` – Feature importance plot  

### Tech Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, SHAP

### Summary
The model cleans the data, handles imbalance, trains classifiers, evaluates performance (accuracy, recall, F1-score), and identifies key factors contributing to fraud detection.
